# Lofi Animation

A simple lofi scene animation created to showcase how to work with [Remotion](https://www.remotion.dev/) and explore programmatic video creation.

## Preview

<img src="lofi-scene.gif" alt="Lofi Scene Animation" width="200" />

## Why This Project

This project showcases how to work with [Remotion](https://www.remotion.dev/) for creating animations programmatically with React.

## Assets

All assets in this project were drawn by me. You can view the design files on [Figma](https://www.figma.com/design/N3IUZAM8aHFVVHRDhI4VEq/Lofi-remotion?node-id=0-1&t=xXN4T0rJkQKfqFls-1).

## Installation

```bash
bun install
```

## Development

Preview the animation in Remotion Studio:

```bash
bun run dev
```

## Render

Render the final video:

```bash
npx remotion render mobile-scene out/lofi-scene.gif
```

The video will be saved in the `out` directory.
